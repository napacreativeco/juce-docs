bool ausdk::AUBase::ValidFormat ( AudioUnitScope  inScope,

const AudioStreamBasicDescription &  inNewFormat 
)
virtual

Will only be called after StreamFormatWritable has succeeded. Default implementation requires non-interleaved native-endian 32-bit float, any sample rate, any number of channels; override when other formats are supported. A subclass's override can choose to always return true and trap invalid formats in ChangeStreamFormat.
