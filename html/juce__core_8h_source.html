371 a linker error if not all of your compile units are consistent in the options that they
372 enable before including JUCE headers. The reason this is important is that if you have
373 two cpp files, and one includes the juce headers with debug enabled, and another does so
374 without that, then each will be generating code with different class layouts, and you'll
375 get subtle and hard-to-track-down memory corruption!
